# Driver_Churn_Analysis
Driver Churn Analysis | SQL + Power BI + Cohort Analysis
# Project Overview
This project analyzes driver churn behavior for a ride-hailing platform similar to Rapido or Uber using simulated operational data.
Objective is to identify:
1.why drivers leave the platform,
2.when churn risk increases,
3.which driver segments are most vulnerable,
4.and how retention can be improved using data-driven insights.

The project combines:
SQL analytics
cohort retention analysis
behavioral analysis
KPI tracking
interactive Power BI dashboards
churn-risk segmentation

to simulate real-world business decision-making in mobility and gig-economy operations.
